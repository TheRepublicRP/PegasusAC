This is a quick installation tip to help ensure Pegasus is set up correctly.
Please make sure to read the documentation to avoid common setup issues.

If you choose not to read the full documentation, at the very least, ensure the following:

1.Webhooks are properly configured
2.Your license identifiers (Discord ID, FiveM ID, or license key) are added to the permissions.json file under the God group

Only after completing these steps should you join the server.
